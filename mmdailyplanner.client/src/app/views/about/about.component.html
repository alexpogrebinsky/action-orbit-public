About:

ActionOrbit is a sophisticated multi-tenant application where each tenant's data remains isolated, ensuring privacy and security. This shared infrastructure model not only reduces costs but also simplifies scalability.

The application's architecture is built to handle extensive error handling, data validation, and user authentication, making it highly scalable and secure.

Angular Material is extensively used to create intuitive and responsive user interfaces, particularly for displaying task grids. Users benefit from in-depth analytics of their tasks and work history, empowering them to manage their productivity effectively.

This open application lays a strong foundation for future scaling and feature additions, ensuring it can grow and adapt to meet evolving user needs.

Technology Stack:

Front-End:

  • Angular: A robust framework for building dynamic web applications. Known for its powerful data binding and dependency injection.
  • Ngx-Charts: A charting library for Angular, providing beautiful and interactive charts.
  • RxJs: A library for reactive programming using Observables.
  • Angular Material: A UI component library following Google's Material Design guidelines.

Back-End:

  • C# .NET Core Web API: A powerful framework for building RESTful APIs, cross-platform, high-performance, and scalable.

ORM:

  • Entity Framework: An ORM that simplifies data access by allowing developers to work with database objects using .NET objects.

Database:

  • SQL Server: A relational database management system known for its reliability, scalability, and comprehensive data management features.

Testing:

  • xUnit: A popular unit testing framework for .NET applications, known for its simplicity and extensibility.

Logging:

  • SeriLog: A logging library for .NET applications that provides structured logging capabilities.

Features:

  • Multi-Tenant Infrastructure: Supports multiple tenants with isolated data.
  • Authentication and Session Management: Secure access and seamless user experience.
  • Input and Data Validation: Ensures user inputs are correct and secure.
  • Error Handling and Logging: Comprehensive error handling mechanisms and logging using SeriLog.
  • User Registration: Captures user details with validation and secure data recording.
  • Task Management Workflow: Task creation, modification, deletion, analysis, and color-coded priority levels.
